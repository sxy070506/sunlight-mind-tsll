## 🎨 整体风格建议

- **色调**：暖橘 + 薄荷绿 + 天空蓝 + 浅樱粉(治愈、阳光、明媚)
- **风格关键词**：#柔和 #治愈 #阳光 #扁平插画 #莫兰迪低饱和
- **避免**：过暗、过饱和、惊悚、低质量

## 💡 快速获取图片的途径

| 来源 | 网址 | 适合 |
|------|------|------|
| Unsplash | unsplash.com | 高质量风景照(免费商用) |
| Pixabay | pixabay.com | 风景 + 插画 |
| Iconfont | iconfont.cn | 图标和小插画 |
| Undraw | undraw.co | 治愈风扁平插画(可换色) |
| AI 生成 | DALL·E / SD / 文心一格 | 自定义头像/插画 |

## ⚙️ 图片不放也能用

代码中**全部使用 CSS 渐变 + emoji 作为占位**,即使一张图都不放,效果也是完整治愈的。
后期只需把图片放入 `assets/images/` 目录,**无需修改任何代码**,自动生效。
